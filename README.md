# PythonFlaskTutorial
Flask Tutorial in Visual Studio Code

1. Run by PowerShell:\
./env/Scripts/Activate.ps1\
$env:FLASK_APP = "hello_app.webapp"\
python -m flask run

2. Create reqirement.txt\
python -m venv .venv\
./.venv/Scripts/Activate.ps1\
pip install -r requirements.txt

3. Install requirement.txt\
pip install -r requirements.txt
